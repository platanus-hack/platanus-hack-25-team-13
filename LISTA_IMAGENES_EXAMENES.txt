================================================================================
ESPECIFICACIONES TÉCNICAS
================================================================================

FORMATO:
  - Extensión: .png o .jpg
  - Nombre de archivo: imagen.png (o imagen.jpg)

TAMAÑO RECOMENDADO:
  - Mínimo: 800x600px
  - Ideal: 1200x900px o mayor
  - Máximo: 2000x1500px (para optimizar carga)

FONDO Y ESTILO:
  - Exámenes de imagen (radiografía, ecografía, tomografía, resonancia):
    * Fondo oscuro (#1a1a1a o negro)
    * Contraste alto
    * Estilo médico realista
  
  - Exámenes de laboratorio:
    * Fondo claro (blanco o #f5f5f5)
    * Texto legible
    * Formato de tabla o gráfico
  
  - Electrocardiograma:
    * Fondo oscuro o claro (según preferencia)
    * Ondas claramente visibles
    * Grid de referencia opcional

CALIDAD:
  - Resolución: Alta (mínimo 300 DPI si es posible)
  - Compresión: Balanceada (calidad vs tamaño)
  - Transparencia: No necesaria (fondo sólido)

================================================================================
NOTAS IMPORTANTES
================================================================================

1. Todas las carpetas ya están creadas en: public/examenes/
2. Cada carpeta debe contener al menos una imagen llamada "imagen.png"
3. Puedes agregar múltiples imágenes en cada carpeta (imagen1.png, imagen2.png, etc.)
4. El sistema buscará automáticamente "imagen.png" en la ruta especificada
5. Si una imagen no existe, el sistema mostrará un placeholder o error
